Resources (แหล่งข้อมูล)

 

          แหล่งข้อมูลของระบบการเรียนการสอนผ่านเครือข่ายเป็นส่วนการทำงานที่ไว้สำหรับผู้ใช้ที่มีหน้าที่ในการเรียนการสอนในวิชานั้นๆได้เข้ามาใช้ข้อมูลที่ได้จัดเก็บไว้ในแหล่งข้อมูลนั้นๆ โดยสามารถแบ่งผู้ใช้ออกเป็น

          w  อาจารย์  ทำหน้าที่ในการ Upload  ข้อมูลการเรียนการสอนมาเก็บไว้ในระบบ

          w  นักเรียน  ทำหน้าที่ในการ Download ข้อมูลการเรียนการสอนในรายวิชาที่นักเรียนได้ลงทะเบียนเรียนไปประกอบในการเรียน

          w  ประเภทของข้อมูลที่สามารถ Upload ขึ้นไปบนระบบ ได้แก่ URL คือ ชื่อของ URL ที่เชื่อมโยงไปยัง Web site ที่มีข้อมูลประกอบการเรียนการสอน และข้อมูลเอกสารอิเล็กทรอนิคต่างๆ เช่น Document ต่างๆ ที่สร้างจากโปรแกรม MS.office และโปรแกรมอื่นๆ

 

ขั้นตอนการสร้างแหล่งข้อมูล

          1. ผู้ใช้สามารถสร้างสร้างแหล่งข้อมูลได้โดยคลิกที่รูป    ที่ปรากฎอยู่ตรง Administrator Menu ในหน้าแรกของ Course จะปรากฎหน้าจอ ดังรูปที่ 120

 

รูปที่ 1  แสดงขั้นตอนการสร้างแหล่งข้อมูล

 

 

w  ที่ช่อง Create new : ให้ระบุชื่อสร้างแบบทดสอบหรือแบบสอบถาม

          w  เลือกที่หัวข้อ Resources

          w  คลิกที่ปุ่ม Create

 

     2. เมื่อคลิกที่ปุ่ม Create ระบบจะสร้างแหล่งข้อมูลให้ และจะปรากฎชื่อและสัญลักษณ์ของแหล่งข้อมูลแสดงอยู่บนเมนูด้านซ้ายมือของจอภาพ ดีงรูปที่ 121

 

รูปที่ 2  แสดงขั้นตอนการสร้างแหล่งข้อมูล

 

องค์ประกอบของแหล่งข้อมูล

 

New Item

         New Item มีลักษณะเป็น Drop Down list  ดังรูปที่ 3 และ New Item ใช้สำหรับเพิ่มข้อมูลใหม่เข้าไปในรายวิชานั้นๆ ซึ่งประกอบด้วย การเพิ่ม Folder, URL, File

 
 


           

รูปที่ 3  แสดงองค์ประกอบของแหล่งข้อมูล

 

     w  Folder เป็นกล่องที่ใช้สำหรับเก็บข้อมูล เพื่อแยกเก็บข้อมูลให้เป็นหมวดหมู่ตามความความต้องการของผู้ใช้

     w  URL  เป็นชื่อของ URL ที่เชื่อมโยงไปยัง Web site ที่มีข้อมูลประกอบการเรียนการสอน

     w  File  เป็นส่วนที่เชื่อมโยงไปยังเอกสารอิเล็คทรอนิคต่างๆ ที่ผ฿ใช้ได้ Upload ขึ้นสู่ระบบ

 

     2. เมื่อคลิกเลือกรายการใน New Item ระบบจะแสดงหน้าจอให้เลือก Upload ข้อมูลเข้าสู่ระบบตามรูปแบบต่างๆ ดังนี้

 

Add Folder (กล่องข้อมูล)

     ใช้สำหรับเก็บข้อมูลที่ Upload เข้ามาในระบบ  เพื่อแยกจัดเก็บข้อมูลให้เป็นหมวดหมู่ ตามประเภทของข้อมูลหรือความต้องการของผู้ใช้  ซึ่งมีรายละเอียดดังรูปที่ 123

 

รูปที่ 4  แสดงการ Upload Folder

 

     w  หากต้องการเพิ่ม Folder เข้าสู่ระบบให้ระบุชื่อของ Folder ที่ช่อง Folder Name : แล้วคลิกที่ปุ่ม  เพื่อเพิ่ม Folder ใหม่เข้าสู่ระบบ

 

Add URL (การสร้าง Link)

          ใช้สำหรับสร้าง Link เพื่อเชื่อมต่อไปยัง Web site ที่มีข้อมูลประกอบการเรียนการสอนในรายวิชาต่าง  ซึ่งมีรายละเอียดดังรูปที่ 124

 

รูปที่ 5  แสดงการ Upload URL

 

     w  URL Name :  ชื่อของ URL ที่ผู้ใช้ตั้งขึ้นเพื่อให้แสดงในแหล่งข้อมูล

     w  URL  : ชื่อของ URL จริงที่เชื่อมโยงไปยัง Web site ที่มีข้อมูลจริง

       Show Type : เป็นการเลือกรูปแบบที่ให้แสดงผลบนหน้าจอของ URL ที่ได้ทำการเชื่อมโยงไว้ ซึ่งจะมี 3 รูปแบบ คือ

1.     Show URL in same window  คือ แสดงผลในขอบเขตที่จัดไว้

2.     Show URL in new window  คือ แสดงผลในอีกหน้าต่างของ window

3.     Show URL in full screen window คือ แสดงผลแบบเต็มจอ

 

Add File (เพิ่มข้อมูล)

          เป็นส่วนที่ใช้สำหรับเพิ่มข้อมูลหรือ File ต่างๆที่เกี่ยวข้องกับรายวิชาที่สอนหรืองานทำในแต่ละรายวิชาเข้าสู่ระบบ  ซึ่งมีขั้นตอนการใช้งานดังต่อไปนี้

 

รูปที่ 6  แสดงการ Upload File

 

w  File name : เป็นชื่อของแฟ้มข้อมูลที่ขึ้นเพื่อให้แสดงในหน้าของแหล่งข้อมูล

w  File Upload : ชื่อของแฟ้มข้อมูลที่จะทำการ  Upload ขึ้นไปในระบบโดยมีเงื่อนไขว่าชื่อต้องเป็นภาษาอังกฤษเท่านั้น

w  Show Type : เลือกรูปแบบการแสดง File ที่ Upload ขึ้นไปในระบบ

1.    Show URL in same window  คือ แสดงผลในขอบเขตที่จัดไว้

2.     Show URL in new window  คือ แสดงผลในอีกหน้าต่างของ window

3.     Show URL in full screen window คือ แสดงผลแบบเต็มจอ

w  คลิกที่ปุ่ม   เพื่อ Upload ข้อมูลเข้าในระบบ

 

การแก้ไขข้อมูล

          ใช้สำหรับแก้ไขข้อมูลที่ทำการ Upload เข้ามาในระบบ เมื่อผู้ใช้ Upload ข้อมูลเสร็จแล้วจะแสดงหน้าจอดังรูปที่ 126

รูปที่ 7  แสดงขั้นตอนการแก้ไข File

 

w        เมื่อปรากฎหน้าจอดังรูปที่ 127 ให้ผู้ใช้คลิกที่รูป หลังชื่อ File ที่ต้องการจะปรากฎหน้า

  จอดังรูปที่ 127 และผู้ใช้สามารถแก้ไขข้อมมูลได้ตามขั้นตอนต่อไปนี้

 

รูปที่ 8  แสดงขั้นตอนการแก้ไข File

 

          1. File Name : แก้ไขชื่อ File ที่ต้องการ

             w  คลิกที่ปุ่ม  เพื่อแก้ไขชื่อ File

          2. File Upload : เลือก File ที่ต้องการ Upload เข้าไปแทนที่ File เดิม

             w  Show Type : เลือกรูปแบบการแสดงข้อมูล

             w  คลิกที่ปุ่ม  เพื่อยกเลิกการ Upload File

   w  คลิกที่ปุ่ม  เพื่อ Upload File เข้าไปในระบบ File ที่ทำการ Upload เข้าไปใหม่จะเข้าไปแทนที่ File เดิม

          3. Move to folder : เลือก Folder ที่ต้องการย้านข้อมูลนั้นๆไปจัดเก็บ

              w  คลิกที่ปุ่ม  เพื่อเพื่อย้ายข้อมูล

          4. คลิกที่ปุ่ม  เพื่อลบข้อมูล
