Homework ( การบ้าน )

 

          การบ้านของการเรียนการสอนผ่านระบบเครือข่ายเป็นส่วนของการทำงานที่มีเพื่ออำนวยความสะดวกในการสั่ง  ส่ง  ตรวจการบ้านสำหรับผู้ที่มีหน้าที่ในการเรียนการสอนในวิชานั้นๆได้เข้ามาใช้งาน โดยสามารถแบ่งผู้ใช้ออกเป็น 2 กลุ่มหลัก ๆ คือ

          w อาจารย์  ทำหน้าที่เป็นผู้สั่งการบ้านและตรวจการบ้านที่ผู้เรียนส่งมา โดยสร้างและกำหนดรูปแบบของการบ้านที่รับในรายวิชานั้นๆที่อาจารย์เป็นผู้สอน

          w นักเรียน ทำหน้าที่เป็นผู้ส่งการบ้านตามรูปแบบที่กำหนดไว้ในการเรียนการสอนของรายวิชานั้นๆ ที่นักเรียนได้สมัครเรียน

 

          ส่วนการทำงานแบ่งออกเป็น 3 ส่วน คือ

          1. ส่วนการสร้างและกำหนดรูปแบบการบ้าน : ส่วนนี้ผู้สอนจะเป็นผู้ใช้งาน

          2. ส่วนของการส่งการบ้าน : ส่วนนี้ผู้เรียนจะเป็นคนใช้งาน

          3. ส่วนการตรวจการบ้าน  : ส่วนนี้ผู้สอนเป็นคนใช้งาน

 

1. ขั้นตอนการสร้างและกำหนดรูปแบบของการบ้าน

          1. สร้างชุดการบ้านเพื่อเก็บการบ้านในแต่ละข้อ โดยคลิกที่รูป    ที่ปรากฎอยู่ตรง Administrator Menu ในหน้าแรกของ Course จะปรากฎหน้าจอ ดังรูปที่ 53         

 

รูปที่ 1  แสดงขั้นตอนการสร้างและกำหนดรูปแบบของการบ้าน

 

     2. ให้คลิกเลือกที่ E - Homework จากนั้นให้ตั้งชื่อของการบ้านที่ช่อง Create new : (ในที่นี้จะตั้งเป็นชื่อ Homework ) แล้วคลิกที่ปุ่ม Create เพื่อสร้างเอกสารต่อไป ดังรูปที่ 54

 

รูปที่ 2  แสดงขั้นตอนการสร้างและกำหนดรูปแบบของการบ้าน

 

          3. เมื่อสร้างการบ้านแล้วระบบจะแสดงหน้าจอ Edit Homework ให้ทางจอภาพเพื่อสร้างการบ้านดังรูปที่ 55

 

รูปที่ 3  แสดงขั้นตอนการสร้างและกำหนดรูปแบบของการบ้าน

 

w  Name :  ระบบจะแสดงชื่อการบ้านที่ตั้งไว้ (ในที่นี้คือ Homework)

w  Last date :  ให้เลือกวันที่กำหนดส่งการบ้านในช่องหรือวันสุดท้ายที่สามารถส่งงานได้

w  Info :  พิมพ์รายละเอียดของการบ้าน แล้วคลิกที่ปุ่ม Save ดังรูปที่ 55

w  Active/Inactive :  สถานะของการบ้านหากคลิกให้มีเครื่องหมาย ü ในช่องหนี้ หมายถึง

Active คือให้แสดงการบ้าน แต่ถ้าหากเว้นว่างช่องนี้ไว้จะหมายถึง Inactive คือให้ซ่อนการบ้านชุดนี้ไว้

 

รูปที่ 4  แสดงขั้นตอนการสร้างและกำหนดรูปแบบของการบ้าน

 

 

 

          4. เมื่อทำการสร้างการบ้านเสร็จแล้วจะปรากฎสัญลักษณ์ของ EHomework ชื่อ Homework แสดงอยู่ที่เมนูด้านซ้ายมือ ดังรูปที่ 57

 

รูปที่ 5  แสดงขั้นตอนการสร้างและกำหนดรูปแบบของการบ้าน

 

          5. คลิกที่รูป   ที่อยู่หลังชื่อของการบ้านที่ต้องการแก้ไข ดังรูปที่ 58

 

รูปที่ 6  แสดงขั้นตอนการแก้ไขการบ้าน

 

          6. เมื่อระบบแสดงหน้าจอ  Edit Homework ผู้ใช้สามารถแก้ไขการบ้านได้ตามต้องการ  ตามขั้นตอนที่กล่าวมาแล้วข้างต้น ดังตัวอย่างในรูปที่ 59

 

 

รูปที่ 7  แสดงขั้นตอนการแก้ไขการบ้าน

 

          w  ถ้าต้องการยกเลิกให้คลิกที่ปุ่ม Cancel

 

          7. หากต้องการเพิ่มคำถามให้คลิกที่ชื่อของ Homework ที่เมนูด้านซ้ายมือ ดังรูปที่ 60

 

รูปที่ 8  แสดงขั้นตอนการเพิ่มการบ้าน

 

          8. ระบบแสดงรายละเอียดของคำถาม ดังรูปที่ 61

 

รูปที่ 9  แสดงขั้นตอนการเพิ่มการบ้าน

 

          w   ใช้สำหรับแก้ไขรายละเอียดของหัวข้อการบ้าน

          w    ใช้สำหรับลบหัวข้อการบ้าน

w  คลิกที่  เพื่อแก้ไขคำถาม

w  คลิกที่  เพื่อลบคำถาม

          w  ส่วน Homework Information : เป็นส่วนที่แสดงโจทย์ของการบ้าน และวันที่กำหนดส่งหรือวันสุดท้ายที่สามารถส่งการบ้านนี้ได้  

          w  ส่วน Homework Detail : เป็นส่วนที่แสดงรายละเอียดของการบ้าน ซึ่งมีรายละเอียดดังนี้

                   - No : หมายเลขข้อของการบ้าน

                   - Question : แสดงคำถาม

                   - Attachment / Link : แสดงเอกสารเพิ่มเติมที่หรือ URL ที่เชื่อมไปยัง Web ที่มีข้อมูลเกี่ยวข้องกับคำถามนั้นๆ

                   - Action : แสดงเครื่องมือการทำงาน

          w    ใช้สำหรับสร้างคำถามใหม่

          w    ใช้สำหรับแสดงผลการตรวจการบ้านหรือแสดงคะแนน

 

          9. กรณีที่ต้องการเพิ่มคำถามใหม่ให้คลิกที่  ระบบจะแสดงหน้าจอการเพิ่มคำถามให้ดังรูปที่ 62

 

รูปที่ 10  แสดงขั้นตอนการเพิ่มการบ้าน

 

          w  Question Type : เลือกประเภทของคำถามที่จะให้แสดง ซึ่งมี 3 แบบคือ

                   - Text  แสดงเป็นตัวอักษรธรรมดา

                   -  URL  แสดงเป็น URL ที่เชื่อมโยงไปยัง Web site อื่นที่มีข้อมูลคำถามอยู่

                   -  File   แสดงเป็น File หรือเอกสารอิเล็คทรอนิค

          w  Assignment : ระบุคำถามหรืองานที่ต้องการมอบหมายให้ทำ

          w  Participation : เลือกรูปแบบคำตอบที่ให้นักเรียนส่งกลับมาซึ่งมี 3 แบบคือ

                   -  Typing the answer in textbox  เป็นข้อความตัวอักษรธรรมดา

                   -  Sending URL  ส่งเป็น URL เพื่อเชื่อมโยงไปยัง Website ที่เกี่ยวข้อง

                   -  Sending file  ส่งเป็น File หรือเอกสารอิเล็คทรอนิค

          w  Max Scores :  ระบุคะแนนสูงสุด

          w  คลิกปุ่ม  เพื่อเพิ่มคำถาม

          w  คลิกปุ่ม  เพื่อยกเลิก

 

2. การส่งการบ้าน

 

1.     เมื่อนักเรียน Login เข้าสู่ระบบจะพบหน้าจอหลักของระบบให้เลือกที่เมนู Course

ดังรูปที่ 63

 

รูปที่ 11  แสดงขั้นตอนการส่งการบ้าน

 

          2. เมื่อคลิกที่เมนู Course ระบบจะแสดงรายวิชาต่างๆ ที่นักเรียนได้ลงทะเบียนเรียนตรงเมนูทางด้านซ้ายมือ ให้คลิกเลือกชื่อวิชาที่ต้องการส่งการบ้าน ดังรูปที่ 64 

 

รูปที่ 12  แสดงขั้นตอนการส่งการบ้าน

 

3. เมื่อคลิกเข้าไปในชื่อวิชาที่ต้องการ  แล้วให้คลิกที่ชื่อของการบ้านที่เมนูทางด้านซ้ายมือ  ซึ่งในที่นี้ใช้ชื่อ  Homework  ดังรูปที่ 65

 

รูปที่ 13  แสดงขั้นตอนการส่งการบ้าน

 

4. เมื่อคลิกที่ Homework แล้วระบบจะแสดงหน้าจอ Homework Management ในส่วนที่มีไว้สำหรับนักเรียน ดังรูปที่ 66

 

รูปที่ 14  แสดงขั้นตอนการส่งการบ้าน

 

          5. ให้คลิกที่  เพื่อตอบคำถามและส่งการบ้าน  เมื่อคลิกแล้วระบบจะแสดงหน้าจอ Homework send answer เพื่อให้นักเรียนเข้ามาตอบคำถาม ดังรูปที่ 67

 

รูปที่ 15  แสดงขั้นตอนการส่งการบ้าน

          w  Homework Question : แสดงคำถาม

          w  Answer : ระบุคำตอบ

          w  คลิกที่ปุ่ม  เพื่อส่งการบ้าน

          w  คลิกที่ปุ่ม  เพื่อยกเลิกการส่งการบ้าน

 

          6. เมื่อกดปุ่ม  แล้วระบบจะทำการส่งการบ้านทันทีและจะแสดงสถานะในช่อง Action เป็น  ดังรูปที่ 68

 

รูปที่ 16  แสดงขั้นตอนการส่งการบ้าน

 

          7.  หากผู้ใช้ต้องการเข้ามาแก้ไขคำตอบก็ให้คลิกที่    ระบบจะแสดงหน้าจอการแก้ไขให้ดังรูปที่ 69

 

รูปที่ 17  แสดงขั้นตอนการส่งการบ้าน

 

w  Homework Question : แสดงคำถาม

          w  Answer : ระบุคำตอบที่ต้องการแก้ไข

          w  คลิกที่ปุ่ม  เพื่อส่งการบ้าน

          w  คลิกที่ปุ่ม  เพื่อยกเลิกการส่งการบ้าน

          w  คลิกที่ปุ่ม  เพื่อลบคำตอบนั้นๆ

 

 

3. การตรวจการบ้าน

 

          1. เมื่ออาจารย์คลิกเลือกรายวิชาที่ต้องการตวรจการบ้านแล้วให้คลิกเลือกที่ชื่อการบ้าน ซึ่งในที่นี้ใช้ชื่อ Homework ดังรูปที่ 70

 

รูปที่ 18  แสดงขั้นตอนการตรวจการบ้าน

 

          2. ระบบจะแสดงหน้าจอ Homework Management ในส่วนของอาจารย์ซึ่งมีส่วนการใช้งานต่างๆ ดังต่อนี้

 

รูปที่ 19  แสดงขั้นตอนการตรวจการบ้าน

 

3. คลิกที่ปุ่ม   ระบบจะแสดงหน้าจอ Result ดังรูปที่ 72

 

รูปที่ 20  แสดงขั้นตอนการตรวจการบ้าน

 

          4  ให้คลิกที่รูป  ที่ช่อง Detail ของข้อคำถามที่ต้องการเพื่อตรวจการบ้านในแต่ละข้อที่นักเรียนส่งกลับมา ระบบจะแสดงหน้าจอดังรูปที่ 73

 

รูปที่ 21  แสดงขั้นตอนการตรวจการบ้าน

 

 

          w  คลิกที่ปุ่ม  เพื่อกลับไปยังหน้าแสดงคำถามทั้งหมด

          w  คลิกที่ปุ่ม  เพื่อดูรายละเอียดคำตอบทั้งหมด ดังรูปที่ 2.43

          w  คลิกที่ปุ่ม  เพื่อไปยังหน้าถัดไป

          w  คลิกที่ปุ่ม  เพื่อย้อนกลับไปยังหน้าเดิม

 

 

          5. เมื่อคลิกที่ปุ่ม  ระบบจะแสดงรายละเอียดของคำตอบในข้อนั้นๆ

ดังรูปที่ 74

 

รูปที่ 22  แสดงขั้นตอนการตรวจการบ้าน

 

          w  คลิกที่ปุ่ม  เพื่อย้อนกลับไปยังหน้าก่อนหน้านี้

 

          5. หากต้องการจะกรอกคะแนนให้กับนักเรียน ให้คลิกที่ชื่อของนักเรียนคนที่ต้องการจะกรอกคะแนนให้ ระบบจะแสดงหน้าจอสรุปคะแนนของนักเรียนให้ดังรูปที่ 75

 

รูปที่ 23  แสดงขั้นตอนการตรวจการบ้าน

 

          w  คลิกที่ปุ่ม  เพื่อย้อนกลับไปยังหน้าก่อนหน้านี้

          w  คลิกที่ช่อง Score เพื่อเพิ่มคะแนนให้กับนักเรียน ระบบจะแสดงหน้าจอ Show answer & Score  ดังรูปที่ 76

 

รูปที่ 24  แสดงขั้นตอนการตรวจการบ้าน

          w  ID : หรัส หรือ ชื่อล็อกอินของนักเรียน

          w  Name : ชื่อของนักเรียน

          w  Email : E-mail Address ของนักเรียน

          w  Point :  กรอกคะแนนที่ต้องการ

          w  คลิกที่ปุ่ม  เพื่อบันทึกการให้คะแนน

          w  คลิกที่ปุ่ม  เพื่อปิดหน้าต่าง
