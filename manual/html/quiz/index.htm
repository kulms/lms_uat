แบบทดสอบและแบบสอบถาม (Quiz)

 

1. ขั้นตอนการสร้างแบบสอบถามและแบบสำรวจ

1. ผู้ใช้สามารถสร้างสร้างแบบทดสอบหรือแบบสอบถามโดยคลิกที่รูป    ที่ปรากฎอยู่ตรง Administrator Menu ในหน้าแรกของ Course จะปรากฎหน้าจอ ดังรูปที่ 77        

 

รูปที่ 1  แสดงขั้นตอนการสร้างแบบทดสอบหรือแบบสอบถาม

 

w  ที่ช่อง Create new : ให้ระบุชื่อสร้างแบบทดสอบหรือแบบสอบถาม

          w  เลือกที่หัวข้อ Quiz

          w  คลิกที่ปุ่ม Create

 

          2. ระบบจะแสดงชื่อแบบทดสอบที่ถูกสร้างขึ้นตรงเมนูด้านซ้ายมือ ให้คลิกที่รูป  เพื่อสร้างคำถาม ดังรูปที่ 78

รูปที่ 2  แสดงขั้นตอนการสร้างแบบทดสอบหรือแบบสอบถาม

2. Edit Preferences

          โปรแกรมในส่วนจะใช้สำหรับแก้ไขรูปแบบและข้อมูลในแบบทดสอบหรือแบบสอบถาม ซึ่งมีขั้นตอนในการใช้งานดังต่อไปนี้

 

          1. เมื่อคลิกที่รูป  หลังชื่อแบบทดสอบหรือแบบสอบถามจากเมนูด้านซ้ายมือแล้วระบบจะแสดงหน้าจอ  Preferences ให้ทางจอภาพเพื่อเพิ่มรายละเอียดของแบบทดสอบหรือแบบสอบถาม  ซึ่งสามารถกำหนดรูปแบบของแบบทดสอบหรือแบบสอบถามได้ตามหมายเลขที่ระบุในรูปที่ 79  ซึ่งมีรายละเอียดดังนี้

          (1)  Quiz Name / ชื่อ : กำหนด ชื่อของแบบทดสอบ

(2)  Quiz  or Servey / แบบทดสอบ หรือ แบบประเมิน : เลือกว่าจะสร้างแบบทดสอบหรือแบบสอบถาม

          (3)  Last Date / วันสุดท้ายในการทำข้อสอบ : ระบุวันที่สร้างแบบทดสอบหรือแบบสอบถาม

          (4) Java Script validation : ระบุข้อความที่ต้องการให้แจ้งเตือนในการตอบแบบทดสอบ

          (5) Multiple Chioce Format / ช่องคำตอบปรนัย : ระบุชนิดของช่องคำตอบ

          (6)  Randomize question / สุ่มคำถาม : กำหนดให้เป็นแบบ Random หรือเรียงตามลำดับ

          (7)  View answer/ ดูคำตอบ  : กำหนดให้แสดงคำตอบ

          (8) View solution / ดูวิธีทำ : กำหนดให้แสดงวิธีทำหรือสูตรในการทำแบบสอบถาม

          (8)  Sereral Times / จำนวนครั้งในการทำข้อสอบ: กำหนดรูปแบบการตอบว่าให้ทำได้เพียงครั้งเดียวหรือหลายครั้ง

          (9) Select Matching Item / เลือกข้อสอบแบบจับคู่ : กำหนดรูปแบบการใช้ข้อสอบว่าต้องการใช้ ข้อสอบแบบจับคู่หรือไม่

          (10) Select Timer/ ต้องการจับเวลาในการทำแบบทดสอบหรือไม่ : กำหนดการจับเวลาในการทำแบบทดสอบ

(11) ต้องการนำแบบทดสอบชุดนี้เป็นตัวตัดเกรดหรือไม่ : กำหนดว่าต้องการเลือกแบบทดสอบนี้เป็นตัวตัดเกรด

          (9)  Nr of questions for each occasion : กำหนดจำนวนข้อในการตอบ

          (1) Select Background color/ สีที่ต้องการใช้ในเป็นพื้นหลังของคำถาม : กำหนดสี พื้นหลังของคำถาม

          (10) Start page : พิมพ์คำชี้แจงหรือกำหนดคำเริ่มต้นในการทำแบบทดสอบ เช่น Start

          w  เมื่อกำหนดรูปแบบต่าง ๆ ครบตามลำดับแล้วให้คลิกที่ปุ่ม  /  เพื่อสร้างหรือแก้ไขคำถามต่อไป

w        ผู้ใช้สามารถคลิกที่ปุ่ม  เพื่อ Reset ค่าใหม่

 

รูปที่ 3  แสดงขั้นตอนการแก้ไขแบบทดสอบหรือแบบสอบถาม

          3. เมื่อคลิกที่ปุ่ม   /   แต่ในกรณีที่ผู้ใช้ เลือกแบบทดสอบให้เป็นชนิดแบบประเมิน ระบบจะแสดงข้อความเพื่อยืนยันการสร้างแบบทดสอบหรือแบบสอบถาม ให้คลิกที่ปุ่ม OK ดังรูปที่ 80

 

รูปที่ 4  แสดงขั้นตอนการแก้ไขแบบทดสอบหรือแบบสอบถาม

 

 

         

 

3. Add new question

 

          โปรแกรมในส่วนนี้ใช้สำหรับเพิ่มคำถามเข้าไปในแบบทดสอบหรือแบบสอบ ซึ่งจะมีรูปแบบของคำถามให้เลือกในเมนู

 

 

        3.1  แบบ Multiple Choice

                เป็นสร้างแบบทดสอบหรือแบบสอบถามในลักษณะที่มีตัวเลือกของคำตอบหลายข้อ (Choice) แต่จะมีคำตอบที่ถูกต้องเพียงข้อเดียวหรือตามที่อาจารย์กำหนด  ซึ่งมีขั้นตอนการสร้างดังต่อไปนี้ 

          w  คลิกเลื่อกที่เมนู Multiple Choice / แบบตัวเลือก ที่เมนูหลักด้านบนของจอภาพ ระบบจะแสดงหน้าจอ ดังรูปที่ 81

 

 

รูปที่ 5  แสดงการเพิ่มคำถามในแบบทดสอบแบบ Multiple Choie

 

w  Question : ใช้สำหรับพิมพ์คำถามที่ต้องการ

          w  File Attach : ใช้สำหรับแทรก File ที่ต้องการใช้ในคำถามนั้นๆ

          w  Category : ใช้ในการค้นหาคำถามตามแบบหรือประเภทของคำถามต่างๆ

          w  Score : ระบุคะแนนเต็มของข้อสอบในข้อนั้นๆ

w  Alternatives : ใช้ในการพิมพ์คำตอบของคำถามในแต่ละข้อ ซึ่งจะมี Choice ให้เลือก

w  Check if true : ให้คลิกเครื่องหมาย ü ที่ช่อง ¨ ด้านหลังคำตอบที่ถูกต้อง

          w  Solution : ระบุวิธีหาคำตอบ

          w  Comment : แสดงความคิดเห็นหรือข้อเสนอแนะ

          w  เมื่อระบุรายละเอียดต่างๆ เรียบร้อยแล้วให้คลิกที่ปุ่ม   

          w  คลิกที่ปุ่ม  เพื่อReset Form หรือรูปแบบของแบบทดสอบหรือแบบสอบถามใหม่

3.2  True/ False

          เป็นแบบทดสอบหรือแบบสอบถามที่มีการกำหนดคำตอบได้แค่ 2 อย่างคือ ถูกและผิด ซึ่งมีขั้นตอนการสร้างดังต่อไปนี้

 

 

รูปที่ 6  แสดงการเพิ่มคำถามในแบบทดสอบแบบ True/False

 

w  Question : ใช้สำหรับพิมพ์คำถามที่ต้องการ

w        File Attach : ใช้สำหรับแทรก File ที่ต้องการใช้ในคำถามนั้นๆ

w        Answer : ให้ระบุคำตอบ ซึ่งมี 2 อย่างคือ ถูกและผิด ดังรูปที่ 82

          w  Category : ใช้ในการค้นหาคำถามตามแบบหรือประเภทของคำถามต่างๆ

          w  Score : ระบุคะแนนเต็มของข้อสอบในข้อนั้นๆ

w  Solution : ระบุวิธีหาคำตอบด้วยตนเอง

          w  Comment : แสดงความคิดเห็นหรือข้อเสนอแนะ

          w  เมื่อระบุรายละเอียดต่างๆ เรียบร้อยแล้วให้คลิกที่ปุ่ม   

          w  คลิกที่ปุ่ม  เพื่อReset Form หรือรูปแบบของแบบทดสอบหรือแบบสอบถามใหม่

 

 

 

 

3.3  Match Item

                   เป็นการสร้างแบบทดสอบในลักษณะการจับคู่กันของคำตอบที่ถูกต้องตามที่โจทย์กำหนด โดยให้ผู้ใช้กรอกรายละเอียดลงในช่องตามที่ปรากฎในหน้าจอของรูปที่ 83

 

รูปที่ 7  แสดงการเพิ่มคำถามในแบบทดสอบแบบ Match Item

 

          w  Category : ใช้ในการค้นหาคำถามตามแบบหรือประเภทของคำถามต่างๆ

          w  T to Question : ระบุคำถามที่ต้องการ

          w  Score of each Q.# : ระบุคะแนนของคำถามแต่ละข้อ

          w  Comment : แสดงความคิดเห็นหรือข้อเสนอแนะ

          w  เมื่อระบุรายละเอียดต่างๆ เรียบร้อยแล้วให้คลิกที่ปุ่ม   

          w  คลิกที่ปุ่ม  เพื่อReset Form หรือรูปแบบของแบบทดสอบหรือแบบสอบถามใหม่

 

3.4  Fill-in-Bank      

                    เป็นการสร้างแบบทดสอบแบบเติมคำในช่องว่างให้ถูกต้อง ตามโจทย์ที่อาจารย์กำหนด ดังรูปที่ 84

 

รูปที่ 8  แสดงการเพิ่มคำถามในแบบทดสอบแบบ Fill-in-Bank

 

w   Question : ใช้สำหรับพิมพ์คำถามที่ต้องการ

w  Category : ใช้ในการค้นหาคำถามตามแบบหรือประเภทของคำถามต่างๆ

          w  Score : ระบุคะแนนเต็มของข้อสอบในข้อนั้นๆ

w  Solution : ระบุวิธีหาคำตอบด้วยตนเอง

          w  Comment : แสดงความคิดเห็นหรือข้อเสนอแนะ

          w  เมื่อระบุรายละเอียดต่างๆ เรียบร้อยแล้วให้คลิกที่ปุ่ม   

          w  คลิกที่ปุ่ม  เพื่อReset Form หรือรูปแบบของแบบทดสอบหรือแบบสอบถามใหม่

 

4.  Set active/ Inactive

          เป็นส่วนที่ใช้ในการกำหนดให้ Quiz ที่สร้างขึ้นมาแสดงทางจอภาพหรืออยู่ในสถานะซ่อนไว้ไม่ให้ผู้ใช้มองเห็น โดยมีขั้นตอนการทำงานดังนี้

 

รูปที่ 9  แสดงขั้นตอนการ Set active/inactive

 

          w  คลิกที่เมนูคำสั่ง Set active/inactive ระบบจะแสดงหน้าจอดังรูปที่ 85

          w  เมื่อเลือกเสร็จให้คลิกที่ปุ่ม
