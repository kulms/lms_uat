กลุ่มสมาชิกและแฟ้มเอกสาร (Group & Folder)

 

          โปรแกรมในส่วนนี้ใช้สำหรับจัดกลุ่มของนักเรียนหรือผู้ใช้ จะช่วยในกรณีในการให้การบ้าน หรือ

quiz ในกลุ่มแต่ละกลุ่ม เฉพาะสมาชิกในกลุ่มนั้นเท่านั้นที่จะสามารถใช้งาน tool ที่ add ลงไปในแต่ละกลุ่มได้ ซึ่งจะช่วยอำนวยความสะดวกในการจัดการกลุ่มผู้ใช้งาน

 

        1. ขั้นตอนการสร้างกลุ่มผู้ใช้งาน ( Group )

1. ผู้ใช้สามารถสร้างสร้างกลุ่มของผู้ใช้งานได้โดยคลิกที่รูป    ที่ปรากฎอยู่ตรง Administrator Menu ในหน้าแรกของ Course จะปรากฎหน้าจอ ดังรูปที่ 45   

 

รูปที่ 1  แสดงขั้นตอนการสร้างกลุ่มผู้ใช้

 

w  ที่ช่อง Create new : ให้ระบุชื่อของ Group ซึ่งในที่นี้ใช้ชื่อว่า  BST

          w  เลือกที่หัวข้อ  Group

          w  คลิกที่ปุ่ม Create

          2. ระบบจะแสดงหน้าจอการ Edit Group member เพื่อเพิ่มสมาชิกในกลุ่มดังรูปที่ 46

 

รูปที่ 2  แสดงขั้นตอนการสร้างกลุ่มผู้ใช้

 

          w  ที่ช่อง Member :  แสดงรายชื่อของสมาชิกในกลุ่ม

          w  ที่ช่อง Other :  แสดงรายชื่อของผู้ใช้ทั้งหมดในรายวิชานั้นๆ

w  คลิกเลือกที่ชื่อของผู้ใช้แล้วคลิกที่ปุ่ม  เพื่อเพิ่มสมาชิก

w  คลิกเลือกที่ชื่อของผู้ใช้แล้วคลิกที่ปุ่ม  เพื่อยกเลิกสมาชิกออกจากกลุ่ม

w  คลิกที่ปุ่ม  เพื่อ Update  สมาชิกในกลุ่ม

 

          3. Tab Filtering Edit Group ใช้สำหรับจัดกลุ่มใหม่ให้กับสมาชิกที่ยังไม่มีกลุ่ม ระบบจะแสดงหน้าจอดังรูปที่ 47

 

รูปที่ 3  แสดงขั้นตอนการสร้างกลุ่มผู้ใช้

 

          w  คลิกเครื่องหมาย ü ที่ช่อง Check All/ Uncheck All เพื่อเช็คจำนวนสมาชิกทั้งหมดในรายวิชานั้นๆ ถ้าไม่คลิกเครื่องหมาย  ü ที่ช่อง  Check All/ Uncheck All  เพื่อไม่ให้เช็คจำนวน

สมาชิกทั้งหมด

          w  คลิกที่ปุ่ม  เพื่อเพิ่มสมาชิกในกลุ่มใหม่

 

        2. การแก้ไข ลบ กลุ่มผู้ใช้

 

          1. เมื่อทำการสร้างกลุ่มผู้ใช้ในระบบเรียบร้อยแล้ว  กลุ่มที่สร้างขึ้นจะแสดงที่เมนูด้านขวามือ

ดังรูปที่ 48 หากผู้ใช้ต้องการแก้ไข Group ให้คลิกที่รูป   เพื่อเข้าไปแก้ไขกลุ่มผู้ใช้

 

รูปที่ 4  แสดงกลุ่มผู้ใช้ที่สร้างใหม่

 

 

 

 

 

          2. เมื่อคลิกที่ปุ่ม  ระบบจะแสดงหน้าจอ Edit / Add กลุ่มของผู้ใช้ ดังรูปที่ 49

รูปที่ 5  แสดงกลุ่มผู้ใช้ที่สร้างใหม่

 

          w  Rename Group : ใช้สำหรับเปลี่ยนชื่อกลุ่ม โดยให้ระบุชื่อใหม่แล้วคลิกที่ปุ่ม

          w  Remove Group : เคลื่อนย้ายกลุ่มของผู้ใช้ไปยัง Folder ที่ต้องการ โดยระบุชื่อกลุ่มแล้วให้คลิกที่ปุ่ม

          w  Delete Group : ลบกลุ่มผู้ใช้งาน โดยการระบุชื่อกลุ่มที่ต้องการลบแล้วคลิกที่ปุ่ม

          w  Create new : ใช้ในการสร้างกลุ่มใหม่และ Folder ต่างๆในการใช้งาน โดยขั้นตอนการสร้างให้ทำเหมือนขั้นตอนแรกที่ได้กล่าวมาแล้วในขั้นต้น

 

        3. การสร้าง Folder

              Folder คือ กล่องที่ใช้สำหรับเก็บเอกสาร เพื่อช่วยในการแยกเก็บเอกสารให้เป็นหมวดหมู่ตามการใช้งานของผู้ใช้ โดยมีขั้นตอนการสร้าง ดังต่อไปนี้

1. ผู้ใช้สามารถสร้างสร้างกลุ่มของผู้ใช้งานได้โดยคลิกที่รูป    ที่ปรากฎอยู่ตรง Administrator Menu ในหน้าแรกของ Course จะปรากฎหน้าจอ ดังรูปที่ 50   

 

รูปที่ 6  แสดงขั้นตอนการสร้าง Folder

          2. เมื่อสร้างเสร็จระบบจะแสดงชื่อ Folder ที่สร้างขึ้นในเมนูด้านซ้ายมือของผู้ใช้ หากต้องการเก็บเอกสารไว้ใน Folder ให้คลิกที่รูป  ด้านหลังชื่อ Folder ที่ต้องการ ดังรูปที่ 51

 

รูปที่ 7  แสดงขั้นตอนการสร้าง Folder

          3. ระบบจะแสดงหน้าจอการ Edit / Add ให้ทางจอภาพผู้ใช้สามารถแก้ไขและลบ folder ที่ต้องการได้

 

รูปที่ 8  แสดงขั้นตอนการสร้างและแก้ไข Folder

          w  Rename folder : ระบุชื่อ Folder ที่ต้องการเปลี่ยนชื่อแล้วคลิกที่ปุ่ม

          w  Delete folder : ใช้ลบ Folder โดยให้ระบุชื่อ Folder ที่ต้องการแล้วคลิกที่ปุ่ม

          w  Create new : ระบุชื่อ Folder หรือเอกสารอย่างอื่นที่ต้องการเก็บใน Folder นั้นๆ แล้วเลือกรายการที่ต้องการสร้าง จากนั้นให้คลิกที่ปุ่ม  

          w  คลิกที่ปุ่ม  เพื่อย้อนกลับไปยังหน้าก่อนหน้านี้
