Web board (กระดานข่าว)

 

          กรดานข่าวเป็นส่วนการทำงานที่มีไว้สำหรับให้ผู้ใช้ได้เข้ามาประกาศข่าวและเพื่อดูข่าวสารต่างๆ เกี่ยวกับรายวิชานั้นๆ เพื่อสมาชิกจะได้รับทราบข้อมูลข่าวสารที่ทันเวลาและถูกต้องตรงกัน

 

          การทำงานแยกออกเป็น 2 ส่วนหลักๆ คือ

          1. ส่วนการสร้างและกำหนดค่าพื้นฐานของกระดานข่าว

          2. ส่วนคำสั่งพื้นฐานของกระดานข่าว

               w  สร้างเพิ่มกระดานข่าวใหม่ใน Web board

               w  ค้นหาข้อมูลที่ระบุในกระทู้ภายในกระดานข่าว

               w  ตั้งค่าให้กับการแสดงผลในกระดานข่าว

 

1. การสร้างและกำหนดค่าพื้นฐานของกระดานข่าว

          1. ผู้ใช้สามารถสร้างสร้างแหล่งข้อมูลได้โดยคลิกที่รูป    ที่ปรากฎอยู่ตรง Administrator Menu ในหน้าแรกของ Course จะปรากฎหน้าจอ ดังรูปที่ 128 

 

รูปที่ 1  แสดงขั้นตอนการสร้างและกำหนดค่าพื้นฐานของกระดานข่าว

 

w  ที่ช่อง Create new : ให้ระบุชื่อของ Web board ซึ่งในที่นี้ใช้ชื่อว่า Web board

          w  เลือกที่หัวข้อ Web board

          w  คลิกที่ปุ่ม Create

 

2. ระบบจะแสดงหน้าจอการเพิ่มรายละเอียดของ Web board ดังรูปที่ 129

 

รูปที่ 2  แสดงขั้นตอนการสร้างและกำหนดค่าพื้นฐานของกระดานข่าว

 

     3. เมื่อคลิกที่ Update แล้วระบบจะแสดง Web board บนเมนูทางด้านซ้ายมือของจอภาพ ดังรูปที่ 130

 

รูปที่ 3  แสดงขั้นตอนการสร้างและกำหนดค่าพื้นฐานของกระดานข่าว

 

     4. หากต้องการแก้ไขชื่อ หัวข้อและรายละเอียดของ Web board ให้คลิกที่รูป  ด้านหลังชื่อ Web board ที่ต้องการตรงเมนูด้านซ้ายมือ  ระบบจะแสดงหน้าต่าง Edit Web board ให้เหมือนรูปที่ 2 ผู้ใช้สามารถแก้ไขรายละเอียดได้ตามต้องการ

     5. หากผู้ใช้ต้องการดูรายละเอียดของ Web board ให้คลิกที่ชื่อของ Web board นั้น ในที่นี้คือ หัวข้อ Web board ระบบจะแสดงหน้าจอของ Web board พร้อมทั้งเมนูในการใช้งานต่างๆ ให้ ดังรูปที่ 131

 

รูปที่ 4  แสดงขั้นตอนการสร้างและกำหนดค่าพื้นฐานของกระดานข่าว

 

2. คำสั่งพื้นฐานของกระดานข่าว

 

คำสั่งพื้นฐานที่ใช้ในกระดานข่าวมี 3  ฟังก์ชั่น คือ

1.      ฟังก์ชั่นการสร้างกระทู้ใหม่ในกระดานข่าว 

2.         ฟังก์ชั่นการค้นหาข้อมูลที่ระบุในกระดานข่าว 

3.         ฟังก์ชั่นการแสดงผลในกระดานข่าว

 

2.1  การสร้างกระทู้ใหม่ในกระดานข่าว

     1. หากผู้ใช้ต้องการเพิ่มกระทู้ใหม่ใน Web board ทำได้โดยการคลิกที่รูป  เพื่อเข้าสู้หน้าจอการเพิ่มกระทู้ใหม่ ดังรูปที่ 132

 

รูปที่ 5  แสดงขั้นตอนการสร้างกระทู้ใหม่

    

w        Subject / หัวข้อ : ให้ระบุหัวข้อของกระทู้ที่ต้องการ

w        Icon of subject/ไอคอนข้อความ : เลือกไอคอนเพื่อแสดงว่า หัวข้อนั้นมีลักษณะเป็นอย่างไร เพื่อความเข้าใจง่าย

w        Fill Message/ข้อความ : ระบุรายละเอียดของกระทู้

w        Smilies / ยิ้มกว้าง : เป็นลูกเล่นเพื่อสื่ออารมณ์ของ ข้อความนั้นๆ

w        Attach Picturn / ภาพประกอบ : ระบุภาพประกอบเมือ หัวข้อนั้นๆมีภาพประกอบ เพื่อทำให้ผู้อ่านสามารถเข้าใจในหัวข้อมากขึ้น

          w  คลิกที่ปุ่ม  หรือ  เพื่อเพิ่มกระทู้เข้าไปในกระดานข่าว

 

 

 

 

 

 

 

 

 

 

          2. เมื่อคลิก Submit แล้วระบบจะกลับมายังหน้าแสดงผลซึ่งจะแสดงกระทู้ทั้งหมด ดังรูปที่ 133

 

รูปที่ 6  แสดงหน้าจอ ที่แสดงกระทู้ทั้งหมด

 

3. เมื่อผู้ใช้ ต้องการที่จะทำการตอบ หรือดู กระทู้ที่ต้องการ สามารถทำได้โดยการคลิกที่หัวข้อกระทู้นั้นๆ โดยหน้านั้นจะทำการแสดงรายละเอียดของกระทู้นั้น และแสดงความคิดเห็นทั้งหมดที่มีการตอบ เพื่อแสดงความคิดเห็น โดยในส่วนนี้ผู้ใช้สามารถทำการแก้ไข ลบ หรือดูข้อมูลผู้ที่ทำการตอบได้ ในส่วน การลบ และแก้ไขนั้น กระทู้ หรือ ความคิดเห็น นั้น ผู้ใช้จะสามารถกระทำได้เฉพาะของตนเองเท่านั้น

 

รูปที่ 7  แสดงหน้าจอ ที่แสดงรายละเอียดของกระทู้

 

          โดยในส่วนนี้ มี คำสั่งพื้นฐาน ที่เพิ่มมาจากหน้าแรก อยู่ 3 ฟังก์ชั่น คือ

                   1.   ฟังก์ชั่นในการ กลับไปหน้าแสดงกระทู้ทั้งหมด

                   2.   ฟังก์ชั่นการสร้างกระทู้ใหม่ในกระดานข่าว 

                   3.  ฟังก์ชั่นเพื่อแสดงความคิดเห็นในกระดานข่าว

     4. โดยที่ผู้ใช้ ที่ต้องการแสดงความคิดเห็น สามารถ คลิกได้ที่  หรือ ทำการแสดงความคิดเห็นได้ที่

 

รูปที่ 8  แสดงหน้าจอ เพื่อแสดงความคิดเห็น

 

2.2  การค้นหาข้อมูลที่ระบุในกระดานข่าว

   

          กรณีต้องการค้นหาข้อมูลใน Web board  โดยระบุข้อมูลเป็นบางส่วน ( Any word ) หรือระบุข้อมูลเป็นคำ (Phrase) ทำได้โดยคลิกที่รูป   แล้วระบบจะแสดงหน้าต่างการค้นหาให้ดังรูปที่ 136

 

รูปที่ 9  แสดงขั้นตอนการค้นหากระทู้

 

     w  ที่ช่อง Search in Web board : ให้คลิกเลือกรูปแบบการค้นหา ดังรูปที่ 136

w  ให้ระบุข้อความหรือกลุ่มคำของข้อมูลที่ต้องการค้นหาแล้วคลิกที่

 

 

2.3  การตั้งค่าการแสดงผลในกระดานข่าว

         

          โปรแกรมในส่วนนี้ใช้สำหรับตั้งค่าการแสดงผลให้กับ Web board สามารถทำได้โดยให้คลิกที่รูป  ระบบจะแสดงหน้าต่างรูปแบบการแสดงผลให้เลือกดังรูปที่ 137

 

รูปที่ 10  แสดงขั้นตอนการกำหนดค่าแสดงผลให้กระทู้

 

     w  Show object & Detail (This option requires java script )  เมื่อต้องการให้แสดงหังข้อและรายละเอียดของกระทู้

     w  Sort descending (Show the last subject first)  เมื่อต้องการให้กระทู้ล่าสุดแสดงขึ้นมาก่อน โดยหัวข้อจะถูกเรียงลำดับตามวันและเวลาที่ตั้งกระทู้

     w  Show the last : เมื่อต้องการให้แสดงเฉพาะรายการกระทู้ ตามจำนวนวันล่าสุด

     w  เมื่อเลิกกำหนดค่าแสดงผลที่ต้องการแล้วให้กดที่ปุ่ม  เพื่อตั้งค่าการแสดงผลให้กับกระทู้

 

หมายเหตุ : เฉพาะเจ้าของกระทู้เท่านั้นที่สามารถ แก้ไข ลบ หรือกำหนดค่าต่างๆ ของกระทู้ได้
